В утилите SQL Server management Studio создать новую базу данных с помощью оператора Create Database, название БД определить, исходя из предметной области.  Программно сделать активной созданную БД с помощью операторв Use.  С помощью оператора Create table, причем самостоятельно определить типы таблиц (родительская или подчиненная), типы полей и их размеры, найти поля Primary key и Foreign Key.
